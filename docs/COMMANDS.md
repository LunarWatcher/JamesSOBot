# Default Plugins


## disable

Disables the bot. Won't respond to messages until `|| enable` is ran by admin.

**Permissions**

- admin
- OWNER
           
**Example**

- `|| sudo disable`

**Shortcuts**

- `disable`

## enable

Enables the bot.

**Permissions**

- admin
- OWNER
           
**Example**

- `|| enable`

**Shortcuts**

- `enable`

## help

Lists commands

**Permissions**

- all
           
**Example**

- `|| help`

**Shortcuts**

- `help`

## info

Gives information about the bot

**Permissions**

- all
           
**Example**

- `|| info`

**Shortcuts**

- `info`

## status

Displays bot status

**Permissions**

- all
           
**Example**

- `|| status`

**Shortcuts**

- `status`

## suicide

Ends the bot's node process.

**Permissions**

- admin
           
**Example**

- `|| sudo restart`

**Shortcuts**

- `die`
- `destroy`
- `selfdestruct`
- `suicide`
- `reboot`
- `restart`
- `just_do_it`
- `shutdown`

## welcome

Welcomes a new user to the room with a message

**Permissions**

- all
           
**Example**

- `|| welecome @JBis`
- `|| welecome JBis`

**Shortcuts**

- `welcome`
