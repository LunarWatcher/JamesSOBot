# addCommand(cmd)

Parameters

Name Type Description
cmd Object
  • Command to add
cmd.name String
  • The name of the command. NOT used to identify a command is being called from a Message (If the name is "test", that doesn't mean that a message calling the command"test" will actually activate this command. If you wan't "test" to call this command, add it to the shortcuts array.)
cmd.args Array
  • A description of the args this command accepts. Only used for descriptive purposes.
cmd.shortcuts Array
  • Keywords that activate this command. If the Message's commandCall matches any of these shortcuts, this command will be activated.
cmd.ignore Boolean
  • Should this command be hidden from the help menu?
cmd.permissions Array
  • Array of groups allowed to use this command. Each entry should line up to group in config.json["users_groups"].
cmd.func CommandFunction
  • The function to call when the command is activated

# remind(user, message, time)

Sets a timer to remind a user a message at a specific time relative to the current time

Alias:

  • timer

Parameters

Name Type Description
user String
  • username of the person to remind, or "me" if you want the bot to remind you
message String
  • the message to be reminded about
time String
  • in how much time, relative to the current time, would you like the remind to occur. Must be in the following format: int [suffix] or int[one letter suffix] Example:

|| remind @JBis Hello. 5 seconds || remind @JBis Hello. 5s

Suffixes:
